============
Installation
============


Stable release
--------------

To install Jupyter 2 Hashnode, run this command in your terminal:

.. code-block:: console

    $ pip install jupyter_2_hashnode

This is the preferred method to install Jupyter 2 Hashnode, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for Jupyter 2 Hashnode can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/tiagopatriciosantos/jupyter_2_hashnode

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/tiagopatriciosantos/jupyter_2_hashnode/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/tiagopatriciosantos/jupyter_2_hashnode
.. _tarball: https://github.com/tiagopatriciosantos/jupyter_2_hashnode/tarball/master
